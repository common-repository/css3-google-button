== Description ==

The Wordpress Plugin *CSS3 GOOGLE BUTTON* is a simple and easy to configure Wordpress plugin which allows you to create CSS3 based Google Buttons inside your blog posts.

**usage** = [GOOGLEBUTTON target="http://liljosh.com" color="orange"]Button Caption[/GOOGLEBUTTON]

* **target** = The target the button will link to. Like the href tag in a hyperlink
* **color** = The color property is required. (orange, blue, gray)
* **search icon** = Type "search icon" in the button caption to get a search icon. (optional)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
